# Créditos

## Criação

**Marcus** — conceito, direção criativa e desenvolvimento.

> marcus esteve aqui, como esteve em Night City.

## Identidade

O universo visual, a cidade pixel art, o cyberdeck Nightwire, o ICEbreaker, os
personagens, a trilha procedural e a linguagem do laboratório foram criados
para este projeto.

O trabalho usa convenções gerais da ficção cyberpunk — redes hostis, ICE,
netrunners e interfaces neurais — sem utilizar imagens, personagens, música ou
assets oficiais de franquias existentes.

## Tecnologias

O projeto utiliza recursos nativos da plataforma web:

- HTML, CSS e JavaScript
- Canvas API
- Web Crypto API
- Web Audio API
- Clipboard API
- Service Worker e Web App Manifest

Não há bibliotecas de interface, serviços externos ou assets de terceiros.
