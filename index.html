ARQUIVO // LINGUAGEM DA RUA

Glossário do Ciberespaço

[ DICIONÁRIO DE CAMPO ]

ICE

Intrusion Countermeasures Electronics. Barreiras digitais que detectam, bloqueiam ou atacam invasores.

Cyberdeck

Computador portátil especializado em invasões, enlaces neurais e navegação pelo ciberespaço.

Netrunner

Operador que atravessa redes hostis, rompe defesas e extrai dados de sistemas protegidos.

Jack in

Conectar-se diretamente a uma rede ou ambiente virtual por uma interface neural ou cyberdeck.

Data shard

Fragmento compacto de informação. Aqui, cada shard contém parte da chave necessária para romper o ICE.

Flatline

Gíria para uma conexão neural encerrada à força. No laboratório, significa que um bug ICE capturou o runner.
